# Release Readiness Plan

## Mục tiêu

Chuẩn hóa checklist trước khi đưa broker adapter vào môi trường cao hơn.

## Checklist

- [ ] Xác định environment matrix.
- [ ] Xác định deployment checklist.
- [ ] Xác định rollback checklist.
- [ ] Xác định incident response checklist.
- [ ] Xác định secrets rotation readiness.
- [ ] Verify monitoring và alerts.
- [ ] Verify health checks.
- [ ] Verify runbook handoff.
- [ ] Verify Paper trước Live.

## Exit Criteria

- [ ] Không release broker nào nếu thiếu Paper signoff.
- [ ] Không bật Live nếu thiếu security/compliance signoff.
