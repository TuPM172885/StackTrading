# Environment Readiness Plan

## Mục tiêu

Theo dõi readiness cho local, test, staging và production.

## Checklist

- [ ] Local development guide.
- [ ] Test environment config.
- [ ] Staging environment config.
- [ ] Production environment config.
- [ ] Kafka bootstrap/topic per environment.
- [ ] Broker endpoint per environment.
- [ ] Secret naming convention.
- [ ] Health check endpoint verified.
- [ ] Log/trace export target verified.

## Exit Criteria

- [ ] Mỗi environment có config source rõ ràng.
- [ ] Không có secret thật trong repo.
